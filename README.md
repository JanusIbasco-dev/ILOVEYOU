# ILOVEYOU - Romantic Website

A dreamy, pastel romantic single-page website built with HTML, CSS, and vanilla JavaScript.

## Files
- `index.html`
- `style.css`
- `script.js`

## Quick Start
Open `index.html` directly in your browser.

If you prefer serving locally, from this folder run:

```powershell
py -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize
- Change the love start date in `script.js` (`loveDate`).
- Replace gallery placeholders in `index.html` with real photos.
- Update the music source URL in the `<audio>` tag in `index.html`.
- Edit footer text in `index.html`.
